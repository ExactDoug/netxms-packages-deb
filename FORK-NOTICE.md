# Fork Notice

This repository is a fork maintained primarily to track the upstream project and to
stage any changes we may contribute back upstream.

Our own downstream implementation, deployment, and environment-specific work are
maintained **separately, in a different repository** — not here — and generally should
not be added to this fork.

If you are modifying this repository, please keep changes scoped to upstream-relevant
fixes or improvements intended for contribution. Implementation- or environment-specific
material does not belong in this fork.
